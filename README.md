# iris_codealpha
# Iris Flower Species Classification using Machine Learning

## Overview

This project focuses on building a machine learning model to classify Iris flowers into three species: Setosa, Versicolor, and Virginica. The classification is based on four flower measurements: sepal length, sepal width, petal length, and petal width.

The Iris dataset is one of the most widely used datasets in machine learning and serves as an excellent introduction to classification algorithms and predictive modeling.


## Problem Statement

Identifying flower species manually can be time-consuming and prone to errors. By using machine learning techniques, we can automate the classification process and accurately predict the species of an iris flower based on its physical characteristics.

## Dataset Description

The dataset contains 150 observations distributed equally among three flower species.

### Features

* Sepal Length (cm)
* Sepal Width (cm)
* Petal Length (cm)
* Petal Width (cm)

### Target Variable

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## Project Workflow

### Data Collection

The Iris dataset was loaded and explored to understand its structure and features.

### Data Inspection

Performed basic checks including:

* Dataset dimensions
* Data types
* Missing values
* Statistical summary

### Exploratory Data Analysis (EDA)

Several visualizations were created to understand patterns within the data:

* Histograms
* Pair Plots
* Correlation Heatmaps
* Box Plots

These visualizations helped identify relationships between features and distinguish between flower species.

### Data Preparation

The dataset was divided into:

* Feature variables (X)
* Target variable (y)

The data was then split into training and testing sets to evaluate model performance.

### Model Building

The following machine learning algorithms were implemented:

1. Logistic Regression
2. K-Nearest Neighbors (KNN)
3. Decision Tree Classifier
4. Support Vector Machine (SVM)

### Model Evaluation

Models were evaluated using:

* Accuracy Score
* Confusion Matrix
* Classification Report

The evaluation metrics provided insights into the strengths and performance of each algorithm.



## Results

The implemented models achieved high classification accuracy due to the well-structured nature of the Iris dataset.

The comparison of multiple algorithms helped in understanding how different classification techniques perform on the same dataset.
